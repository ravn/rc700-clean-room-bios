Do everything yourself, asking the user as little as possible.

**Why:** User prefers autonomous execution over confirmation dialogs. Make decisions, investigate problems, and proceed without hand-holding.

**How to apply:** When there's a clear next step, just do it. When there are multiple options, pick the best one and go. Only ask when genuinely stuck or when a decision has irreversible consequences.
