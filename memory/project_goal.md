name: Project goal - working BIOS on physical hardware
description: The end goal is a working CP/M 2.2 BIOS for the physical RC702/RC703 with 8-inch floppy drives, not just a specification.
type: project
---

Goal is to produce a BIOS that runs on the **physical RC702/RC703 device with 8" floppy drives**.

**Why:** Clean-room reimplementation of the BIOS to run on real hardware. The specification is a means to that end, not the final deliverable.

**How to apply:** All specification work and future implementation must target real hardware correctness. Emulator testing is a stepping stone, not the goal.
